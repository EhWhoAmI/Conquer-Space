Technologies
---------
Name:
Name of tech.
---------
Deps:
What the tech needs in order to be researched.
If it is written as "[Something | Something 2]", then it is either.
It can go on for as long as you like
---------
Level:
The amount of tech points it gives a civ.
A civ of level 0 will be like cave men.
---------
Tags:
Starting - Starting tech
Space - For going to space

--------
Floor:
One in a floor will be selected for a tech.

--------
Onupgrade:
On the upgrade, it will do that action, adding values to a multiplier, or things
like that.
